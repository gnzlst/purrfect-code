# purrfect-code

## Solutions for Google Developers Challenges: Purrfect Code

https://developers.google.com/purrfect-code

https://g.dev/gonzalosoto

## The score considers:

- Total moves
- Braces used
- Semicolons used
- Total lines used
- Bonus batteries

## How to use:

I am using an Array where 0=North, 1=South, 2=East, 3=West
