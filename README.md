# purrfect-code

## Solutions for Google Developers Challenges: Purrfect Code

https://purrfect-code-13ee4.web.app/#game

## The score considers:

- Total moves
- Braces used
- Semicolons used
- Total lines used
- Bonus batteries

## How to use:

I am using an Array where 0=North, 1=South, 2=East, 3=West
